# AI CRM using LangGraph + FastAPI + React

## Project Overview
AI-powered CRM system built using LangGraph workflows, FastAPI backend, and React frontend.

## Features
- AI CRM assistant
- Customer data management
- LangGraph workflow reasoning
- Full-stack integration

## Tech Stack
- Python (FastAPI)
- React.js
- LangGraph
- SQLite

## How to Run

### Backend
cd ai-backend  
uvicorn main:app --reload  

### Frontend
cd frontend  
npm install  
npm start  

## Note
Do not upload .env file to GitHub (contains API keys)
